Данное приложение читает предоставленный файл .csv с ';' в качестве разделителей
либо любой текстовый формат подходящего форматирования.
Затем из полученых данных выводится статистика температуры за указаный период.

-h: Описание функционала приложения (help)
-f <filename.csv>: Входной файл csv для обработки (обязателен)
-m <номер месяца>: Выводить только статистику за указанный месяц
-y <номер года>: Выводить статистику за указаный год. При использовании с -m: 1 месяц указанного года

прим. -y в случае не указания подставляется как текущий год.
